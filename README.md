# fp2gmod (Coffee Potion)

A simple Fabric mod for Minecraft 1.21.11 that adds a Coffee item to the game

## Features
- Adds a "Coffee Potion" item
- Adds a "Caffeinated" effect

## Installation
1. Install Fabric Loader
2. Download the mod jar file
3. Place the jar file and the Fabric API into your game's "mods" folder (...\\AppData\\Roaming\\.minecraft\\mods)
4. Launch the game

## Building Jar File from Source
To build the project yourself:

1. Clone this repository.
2. Open a terminal in the project folder
3. Run the build command:
    - Windows: gradlew build
    - Mac/Linux: ./gradlew build
4. The output jar file will be generated in "build/libs/"
